# crud-app
CRUD operation using MeteorJs
This app uses the module functionality introduced in Meteor 1.3, but everything else should be applicable to Meteor 1.2 as well.

## Versions
Note that attempts will be made to keep the branches up to date but this isn't guaranteed.

### Running the app

```bash
meteor npm install
meteor add iron:router //For routing
meteor add twbs:bootstrap // For twitter bootstrap
meteor add accounts-password // For meteor login
meteor add accounts-ui // For Meteor login ui
meteor add http // For making http calls
meteor
```

### Scripts

To lint:

```bash
meteor npm run lint
```
